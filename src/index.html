Learning API REST

Using Node.JS, MongoDB, Git & Clouding.Io.

This API is a project developed to learn who to create an API instaled in a server running constantly, retrieving data or saving data in a MongoDB data base.

You can feel free for using it as a Mock Server for your development tests or projects.

Technologies.

  • Language: JavaScript / Node.js
  • Data Base: MongoDB / Atlas
  • Repository: Github
  • Server: Allocate in Clouding.io

Use cases.

User Schema: 

      User {
        name: { String, required },
        surname: { String, required },
        email: { String, required },
        password: { String, required },
        address: { String },
        location: { String },
        province: { String },
        country: { String },
        phone: { String },
        birthdate: { Date },
        timestamps: true,
      }
Users How to get or save users data.
METHOD CALL DESCRIPTION
POST: 217.71.202.202:8000/api/users/ Creates a new user.
GET: 217.71.202.202:8000/api/users/ Get users list.
GET: 217.71.202.202:8000/api/users/size=0?page=0?name=xxxxx Get users list with pagination and filter.
GET: 217.71.202.202:8000/api/users/:id Get an user by Id.
GET: 217.71.202.202:8000/api/users/find/:text Get users list corresponding in name or surname with one text.
DELETE: 217.71.202.202:8000/api/users/:id Delete an user.
PUT: 217.71.202.202:8000/api/users/:id Update an user.

Article Schema:

      Article {
        title: { String, required },
        text: { String, required },
        author: { String },
        timestamps: true,
